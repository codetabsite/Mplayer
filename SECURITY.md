# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| 4.0 | ✅ |
| < 2.0 | ❌ |

## Reporting a Vulnerability

Please **do not** open a public issue for security vulnerabilities.

Report privately via email:
**tuxrealkernel@gmail.com**

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

## Response Time
- Acknowledgement: within 48 hours
- Fix for critical issues: within 7 days

## Scope
MPlayer is fully offline with no internet permission. Relevant areas:
- Local file access and permissions
- MediaStore data handling
- Room database security
- Intent handling

## Out of Scope
- Issues in third-party libraries
- Issues requiring physical device access

Thank you for helping keep MPlayer secure!
